Sync between devices Not connected

Keep this Mac and your iPhone showing the same data automatically. This uses a free database on supabase.com that only you control — Ledger never sends your data anywhere else. One-time setup, about 5 minutes:

  1. Create a free account at supabase.com and start a new project.
  2. Open its SQL Editor, paste the snippet below, and click Run — this makes one small table to hold your data.
  3. Open Project Settings → API. Copy the Project URL, and the anon / public key — on newer projects this is called the Publishable key instead. Never use the "service_role" or "Secret" key here — that one grants full access to your database and must never leave Supabase's dashboard.
  4. Make up any Sync code (like a password) — enter the exact same one on every device you want synced.
  5. Repeat steps 3–4 on your iPhone with the same URL, key, and sync code.
create table ledger_sync ( id text primary key, data jsonb not null, updated_at timestamptz not null default now() ); alter table ledger_sync enable row level security; create policy "public access" on ledger_sync for all using (true) with check (true); grant usage on schema public to anon, authenticated; grant select, insert, update on public.ledger_sync to anon, authenticated;
